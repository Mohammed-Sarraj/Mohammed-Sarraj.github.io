## 🚀 About This Project

My personal site is built to be more than just an online resume. It serves as a:

- **Digital Identity Anchor:** A single, trusted source that connects all my professional profiles (LinkedIn, GitHub, etc.).
- **SEO-Optimized Portfolio:** Showcasing my skills in Angular, ASP.NET Core, and SQL Server with a focus on discoverability.
- **Bilingual Presence:** Fully supports English and Arabic, reflecting my global and local audience.

**Key Technical Features:**
- ⚡ **Performance:** Static HTML/CSS/JS with no external frameworks for maximum speed.
- 📱 **Responsive:** Seamless experience on mobile, tablet, and desktop.
- 🔍 **SEO:** Implements Schema.org (`Person`, `ProfilePage`), canonical URLs, `hreflang` tags, and optimized meta data.
- 🎨 **Modern UI:** Clean, dark-themed design with subtle glassmorphism and scroll animations.
